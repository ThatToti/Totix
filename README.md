> Totix这个库名字来源于自己英文名 Toti, 这个库也是自己业务中碰到的常见实用方法的集合库



## 简介

这个工具库,是一些业务常见的实用方法集合.目前是基于 ES6 开发,所以里面记录的方法,必须基于 babel 来配合使用.



关于兼容更多环境,留到以后持续更新...



## 快速开始

> 安装

    $ npm install -g totix



> 使用

    //目前只支持es6
    
    //支持整个引入统一使用
    import totix from 'totix'
    
    //支持只引入单个方法
    import { someFunc } from 'totix'
